# Быстрый старт в Git Bash

## Проблема: "No such file or directory"

Если вы видите эту ошибку, скорее всего вы находитесь не в той директории.

## Решение

### Шаг 1: Перейти в директорию проекта

Выполните в Git Bash:

```bash
cd "/c/Users/Роман/OneDrive/Desktop/Договора НПД Мазов/03/Проект Марико апп/Код для курсора/mariko_vld"
```

Или используйте относительный путь, если Git Bash открыт в родительской папке:

```bash
cd "Договора НПД Мазов/03/Проект Марико апп/Код для курсора/mariko_vld"
```

### Шаг 2: Проверить, что вы в правильной директории

```bash
pwd
ls -la create_pr.sh
```

Должен отобразиться путь к проекту и файл `create_pr.sh`.

### Шаг 3: Выполнить скрипт

```bash
./create_pr.sh
```

## Альтернатива: Открыть Git Bash прямо в папке проекта

**Самый простой способ:**

1. Откройте проводник Windows
2. Перейдите в папку: `mariko_vld`
3. **Правой кнопкой мыши** в пустом месте папки
4. Выберите **"Git Bash Here"**

Теперь Git Bash откроется сразу в правильной директории!

## Если файл create_pr.sh не найден

Проверьте, что файл существует:

```bash
ls -la *.sh
```

Если файла нет, выполните команды вручную:

```bash
# Убедитесь, что вы в правильной директории
pwd

# Создать ветку
git checkout -b feat/cart-item-quantity-limit

# Добавить изменения
git add .

# Создать коммит
git commit -m "feat: добавлено ограничение максимального количества одинаковых блюд в корзине

- Добавлено поле max_cart_item_quantity в таблицу restaurants (по умолчанию 10)
- Добавлена миграция БД для нового поля
- Обновлены API для создания/обновления ресторанов
- Добавлена настройка maxCartItemQuantity в админ панели
- Реализована проверка лимита при добавлении блюд в корзину
- Добавлены уведомления при достижении лимита
- Добавлена поддержка maxCartItemQuantity в MenuItemComponent
- Кнопка + становится неактивной при достижении максимального количества"

# Отправить изменения
git push origin feat/cart-item-quantity-limit

# Создать PR
gh pr create --base main --head feat/cart-item-quantity-limit --title "feat: ограничение максимального количества одинаковых блюд в корзине" --body-file PR_DESCRIPTION.md --web
```

## Полезные команды Git Bash

```bash
# Показать текущую директорию
pwd

# Показать файлы в текущей директории
ls -la

# Перейти в домашнюю директорию
cd ~

# Перейти в предыдущую директорию
cd ..

# Автодополнение пути (нажмите Tab)
cd /c/Users/Роман/OneDrive/Desktop/Договора[TAB]
```

## Резюме

1. **Перейдите в директорию проекта** с помощью `cd`
2. **Проверьте** что файл `create_pr.sh` существует (`ls -la`)
3. **Выполните** `./create_pr.sh`

Или откройте Git Bash прямо в папке проекта через контекстное меню!
