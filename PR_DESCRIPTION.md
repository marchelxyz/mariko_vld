# feat: добавлено уведомление о передаче собранного меню в ресторан при бронировании

## Описание

Добавлена функция передачи собранного пользователем меню в комментарий при бронировании столика. Теперь при бронировании, если у пользователя есть собранное меню в корзине, оно автоматически передается в ресторан через систему Remarked в виде комментария к бронированию.

## Изменения

### Frontend

- **`frontend/src/features/booking/BookingForm.tsx`**:
  - Добавлено информационное уведомление перед чекбоксом согласия на обработку персональных данных
  - Уведомление отображается только если в корзине есть товары
  - Уведомление информирует пользователя, что собранное меню будет передано в ресторан при подтверждении бронирования
  - Используется компонент `Alert` с иконкой `Info` для красивого отображения
  - Функция `formatCartAsComment` уже существовала и правильно форматирует меню в комментарий

## Как это работает

1. Пользователь собирает меню в корзине
2. При переходе к бронированию столика, если в корзине есть товары:
   - Отображается красивое информационное уведомление перед чекбоксом согласия
   - Уведомление сообщает, что меню будет передано в ресторан
3. При подтверждении бронирования:
   - Меню автоматически форматируется функцией `formatCartAsComment`
   - Добавляется в комментарий к бронированию вместе с другими пожеланиями
   - Комментарий передается в систему Remarked через API

## Формат комментария

Меню форматируется следующим образом:
```
Мой заказ:
• Название блюда 1 (2 шт.)
• Название блюда 2
• Название блюда 3 (3 шт.)
```

## UI/UX

- Уведомление имеет стиль, соответствующий дизайну приложения
- Используется цветовая схема `mariko-primary` для выделения
- Иконка `Info` для визуального обозначения информационного сообщения
- Уведомление расположено логично перед чекбоксом согласия

## Технические детали

- Импортирован компонент `Alert` и `AlertDescription` из `@shared/ui/ui/alert`
- Импортирована иконка `Info` из `lucide-react`
- Уведомление условно рендерится только при наличии товаров в корзине (`cartItems.length > 0`)
- Существующая функция `formatCartAsComment` используется без изменений
