# 🚀 Быстрый старт: ВК Мини-приложение

## Что нужно сделать (пошагово)

### 1️⃣ Зайти на портал разработчиков ВК
👉 [https://dev.vk.com](https://dev.vk.com)

### 2️⃣ Создать приложение
- Нажмите "Создать приложение"
- Выберите тип: **"Мини-приложение"**
- Заполните название и описание

### 3️⃣ Настроить URL приложения
**Где:** Настройки → Мини-приложение

**Что указать:**
- **URL приложения**: `https://vhachapuri.ru` (ваш продакшен)
- **URL для разработки**: `http://localhost:8080` (для локальной разработки)

### 4️⃣ Указать доверенные домены
**Где:** Настройки → Мини-приложение → Доверенные домены

**Что указать:**
```
vhachapuri.ru
*.vhachapuri.ru
```

### 5️⃣ Готово! 🎉

Код уже готов, ничего менять не нужно!

## 🔑 Про ключи (кратко)

### ID приложения
- **Где найти:** Настройки → Основная информация
- **Нужен ли:** Да, но он не секретный
- **Куда вставлять:** Никуда не нужно (код уже готов)

### Секретный ключ
- **Нужен ли:** ❌ НЕТ! Для базовой работы не нужен
- **Когда нужен:** Только если используете VK API на сервере
- **Куда вставлять:** Никуда (если не используете серверные запросы)

## ✅ Чеклист перед запуском

- [ ] Создано приложение на dev.vk.com
- [ ] Указан URL приложения (`https://vhachapuri.ru`)
- [ ] Указан URL для разработки (`http://localhost:8080`)
- [ ] Добавлены доверенные домены (`vhachapuri.ru`, `*.vhachapuri.ru`)
- [ ] Фронтенд задеплоен на продакшен
- [ ] Приложение открывается по ссылке `https://vk.com/app{ваш_id}`

---

## 🧪 Тестирование

### Локально
1. Запустите: `npm run frontend:dev`
2. Установите расширение: [VK Mini Apps Dev Tools](https://chrome.google.com/webstore/detail/vk-mini-apps-dev-tools)
3. Откройте расширение и укажите: `http://localhost:8080`

### На продакшене
1. Задеплойте фронтенд на `vhachapuri.ru`
2. Откройте: `https://vk.com/app{ваш_app_id}`
